Detailed Description

Specifies what is overridden and how, for an API_Element.

Since
Archicad 25
Remarks
In version 25, the cutFillBGPen, coverFillBGPen, draftingFillBGPen, cutSurfaceType and uncutSurfaceType members types are changed to @ref API_OverriddenAttributeOrRGB. In version 25, the hideCutContours and hideUncutContours members were added.
