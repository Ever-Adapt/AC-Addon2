General API Development Kit 27

The General API Development Kit enables software developers to extend the functionality of Archicad.

The following documents give a short overview of the possibilities and requirements.

Add-ons built with this API version are compatible with Archicad 27. You can check what improvements and corrections have been made to the former API versions.

  • New API Features in Archicad 27
  • Compatibiliy

Note, that because of the interface changes significantly release by release, we do not ensure neither interface nor binary compatibility with previous versions. So add-ons compiled with older DevKits cannot be loaded in the Archicad 27.
